### 纹理背景图

**需求**

PPT 增加一个带纹理的背景图。

**预期效果图**

![演示产品图](./assets/ppt-0.png)

**教程**

浏览器中打开网址 [trianglify.io](https://trianglify.io/)，配置参数即可生成想要的图片。不过导出高清图是收费的，但是你可以把浏览器窗口放大，通过截图的方式获取想要的大小。
